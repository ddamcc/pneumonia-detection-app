<h1>Problem Statement</h1>

The dataset used here is a X-Ray image dataset. It has three directories.
They are Train, Test and Validation.
It has two classes
**Normal : The X-Ray is Normal  
**Pneumonia :  The X-Ray confirms Pneumonia
